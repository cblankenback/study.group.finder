Student API

Endpoint: GET /api/students

Returns a list of all students with their details.

Sample Response: 

[
    {"studentId":1,"studentEmail":"firstaccount@1.com","studentFirstName":"dave","studentLastName":"sal"},
    {"studentId":2,"studentEmail":"secondaccount@2.com","studentFirstName":"Riley","studentLastName":"Andrade"},
    {"studentId":3,"studentEmail":"1@1","studentFirstName":"1","studentLastName":"1"},
    {"studentId":4,"studentEmail":"lois@vogue","studentFirstName":"Dec","studentLastName":"Dec"}
]

Courses for Student API

Endpoint: GET /api/students/{email}/courses

Returns a list of courses for a given student by their email.

Sample Response: 

[
     {"courseId":1,"courseName":"CST-1072","students":[{"studentId":1,"studentEmail":"firstaccount@1.com","studentFirstName":"dave","studentLastName":"sal"}]},
	 {"courseId":2,"courseName":"CST-5571","students":[{"studentId":1,"studentEmail":"firstaccount@1.com","studentFirstName":"dave","studentLastName":"sal"}]}
]
